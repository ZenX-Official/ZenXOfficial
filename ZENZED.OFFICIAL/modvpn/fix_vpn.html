NETWORK_RECV_ERROR
Fixes (Choose 1):
a. Check your internet connectivity, make sure data connection is not turn off or blocked from accessing internet
b. Check selected payload, make sure it is matched with your internet network
c. CDC data of your device / Toggle Airplane mode
d. Check if you are on the latest app & resources version
e. Clear application cache
f. Clear application data (Make sure you have internet connection for the fetching of initial app resources)
g. Update your server & payload resources
h. If you have sim freebies, consume it first using other payloads
i. Reboot your device
NETWORK_EOF_ERROR
Fixes (Choose 1):
a. It is possible that the selected server is already full
b. Double check if you are on the latest app & resources version
c. If errors still occurs, try NETWORK_RECV_ERROR fixes above
TUN_PROP_ERROR
Fixes (Choose 1):
a. All available IPV4 IPs are already being used please try different server
b. If errors still occurs, try NETWORK_RECV_ERROR or NETWORK_EOF_ERROR fixes above
HTTP PROXY STATUS CODE: 302
Fixes: Check your internet connectivity, make sure data connection is not turn off or blocked from accessing internet
HTTP PROXY UNEXPECTED EOF
Fixes (Choose 1):
a. It is possible that the proxy of the server is dead/down, contact EUT VPN personnel to fix the issue.
b. You may try to use no proxy option on Experimental Settings or Use Direct Connection
HTTP PROXY STATUS CODE: 501
Fixes: Check selected payload, make sure it is matched with your internet network
AUTHENTICATION FAILED
Fixes (Choose 1):
a. It is possible that your app version is not on the latest version, update it on Playstore to fix the problem. For OTA, check carefully your credentials inputted on EUT VPN client.
b. Sometimes this error occurs during the server routine maintenance [00:00 / 6:00 / 12:00 / 18:00 (GMT +8)]
c. If you are using secret server double check your access expiration
CONNECTING / WAITING FOR PROXY OR SERVER ERROR
Fixes: It is possible that the selected server is dead/down, sometimes reconnecting solved the issue. If problem still persist you can contact EUT VPN personnel to fix the issue.
BAD PRIVATE KEY
Fixes: Update your app to the latest version on Playstore. For OTA, check carefully your inputted private key from your account details.
CERTIFICATE VERIFICATION FAILED
Fixes: Correct your device date & time
